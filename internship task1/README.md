A Python-based Command Line Interface (CLI) application to manage a shopping cart efficiently. This project demonstrates core Python concepts including list manipulation, dictionary handling, loops, and conditional logic.

##  Project Overview
This system allows users to act as a store manager or a customer to manage product entries in a cart. It supports CRUD (Create, Read, Update, Delete) operations dynamically.

**Developed by:** Inayat Ali  
**Language:** Python 3.x

## âœ¨ Features
* **Add Product:** Add new items with a unique ID, Name, Price, and Quantity.
* **Duplicate Check:** Prevents adding products with existing IDs.
* **View Cart:** Displays all items with a calculated individual bill and a Grand Total.
* **Search Functionality:** Find specific products quickly by their unique ID.
* **Update Product:** Modify the Price or Quantity of existing items.
* **Delete Product:** Remove items from the cart seamlessly.
* **Error Handling:** Robust validation for invalid inputs (e.g., entering text instead of numbers).
